Les étapes pour exécuter le code :
1) importer le dossier backend dans eclipse 
2) créer une BD sous le nom "car_rental  dans phpmyadmin
3) Exécuter chaque microservice dans le backend (run as --> java application)
4) importer le fichier "cars.sql" dans la table cars et de meme le fichier locations.sql dans la table locations du phpmyadmin
5) maintenant il faut ouvrir le dossier frontend dans Visual Studio Code par exemple
6) aller au terminal du VS Code et tapez npm install pour installer le node_modules
7) pour exécuter le frontend tapez la commande ng serve
8) dans le navigateur tapez dans la plage d'adresse " http://localhost:4200 "
